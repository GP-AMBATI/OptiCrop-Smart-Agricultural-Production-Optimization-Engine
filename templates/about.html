About OptiCrop

OptiCrop is built to help farmers and researchers make intelligent crop decisions using state-of-the-art machine learning. The system analyzes soil nutrients, temperature, humidity, pH and rainfall to recommend the most suitable crop for cultivation.

Core Capabilities
  • Automated crop recommendations with confidence scores.
  • Interactive dashboard with model comparison charts.
  • Prediction history, CSV export, and PDF report generation.
  • Responsive design for desktop and mobile access.
System Design

The application combines a Flask backend, SQLite persistence, and a trained ML model stored using Joblib. It is ready for deployment on modern Python hosting platforms.

Project Goals

Empower agricultural planning, reduce crop risk, and provide actionable crop management guidance.

Dataset: Crop recommendation dataset.

Storage: SQLite database.

Deployment-ready: Render, Railway, PythonAnywhere.
